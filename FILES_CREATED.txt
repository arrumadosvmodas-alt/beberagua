   - email (unique)
   - weight, age

2. water_logs
   - id (UUID, PK)
   - user_id (FK)
   - amount_ml
   - timestamp
   - time_of_day (enum)

3. reminder_settings
   - id (UUID, PK)
   - user_id (FK, unique)
   - start_time, end_time
   - interval_minutes (default 60)
   - enabled (default true)

4. water_stats
   - id (UUID, PK)
   - user_id (FK)
   - date, total_consumed, goal
   - percentage_met

5. notification_prefs
   - id (UUID, PK)
   - user_id (FK, unique)
   - push/email/phone flags

6. fasting_sessions
   - id (UUID, PK)
   - user_id (FK)
   - start_time, end_time, notes
