🚀 GETTING STARTED (5 MINUTOS)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

STEP 1: Open GETTING_STARTED.md
   → Tem um checklist completo de configuração

STEP 2: Create Supabase Account (2 min)
   → https://supabase.com → New Project

STEP 3: Run SQL Script (1 min)
   → Copie script de SETUP.md
   → Cole em Supabase > SQL Editor
   → Execute

STEP 4: Copy Credentials (1 min)
   → Supabase > Settings > API
   → Copie URL e chaves
   → Cole em .env.local

STEP 5: Test Locally (1 min)
   → npm run dev
   → Abra http://localhost:3000
   → Crie conta e teste!

P: Preciso de cartão de crédito para Supabase?
R: Não, o plano Free é suficiente para desenvolvimento.

P: Posso usar outro banco de dados?
R: Sim, mas precisaria reescrever o código Supabase.

P: Como adiciono notificações?
R: Fase 2 com Railway Cron + Firebase Cloud Messaging.

P: Posso fazer deploy em outro lugar?
R: Sim! Vercel é recomendado, mas funciona em Railway, Netlify, etc.

P: Quanto custa para colocar em produção?
R: Começa gratuito com Vercel Free + Supabase Free. Escala com uso.
